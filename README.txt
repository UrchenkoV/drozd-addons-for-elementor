== Description ==

== Installation ==
How to Install Drozd Addons for Elementor?

Before you begin, make sure that you have installed & activated [Elementor](https://wordpress.org/plugins/elementor/)  Plugin on your website. Follow these steps below to install Drozd Addons for Elementor.

Step 1: Go to WordPress dashboard and go to wp-admin – > Plugins – > Add New. Then, in the search box, type the keyword “Drozd-Addons for Elementor”. Then click “Install Now”.

Step 2:  After the installation is completed, click on the “Activate” button and “Drozd – Addons for Elementor” will be activated.

Step 3: Once you have successfully completed Step 2 you can use our addons.
